# Сетевой анализ романа "Война и мир"

# Описание проекта

## Автор: Егоров Михаил Олегович

В данном проекте проводится сетевой анализ взаимодействий персонажей романа Льва Толстого "Война и мир".

На основе XML-разметки текста извлекаются реплики персонажей и адресаты этих реплик. По этим данным строится сеть взаимодействий, где:

вершины графа представляют персонажей  
ребра отражают речевые взаимодействия между персонажами  

Далее к полученному графу применяются методы анализа социальных сетей для изучения структуры взаимодействий между героями произведения.

Анализ реализован на языке R с использованием библиотек xml2, tidyverse и igraph.

# Источник данных

В качестве источника используется XML-разметка романа "Война и мир".

Файл:

War_and_Peace.xml

Файл содержит структурированную TEI-разметку текста, включая информацию о говорящем персонаже и адресатах его реплики.

# Этапы анализа

В рамках проекта выполняются следующие шаги:

1. Парсинг XML-файла и извлечение реплик персонажей.
2. Формирование таблицы взаимодействий между персонажами.
3. Построение ориентированного графа взаимодействий.
4. Преобразование графа в неориентированный взвешенный граф.
5. Расчет основных характеристик сети:
   
число вершин  
число ребер  
плотность сети  
количество компонент связности  
транзитивность  

6. Выделение наибольшей компоненты связности и расчет ее диаметра.
7. Расчет метрик центральности:

degree  
weighted degree  
betweenness centrality  
closeness centrality  

8. Определение точек сочленения графа.
9. Расчет значений k-core.
10. Выделение сообществ персонажей с использованием алгоритма Walktrap.
11. Визуализация структуры сети персонажей.

# Полученные файлы

В результате выполнения скрипта формируются следующие файлы:

speech_data.csv  
Таблица всех речевых взаимодействий персонажей.

top_metrics.csv  
Метрики центральности персонажей.

closeness_largest_component.csv  
Метрика closeness для персонажей в наибольшей компоненте связности.

articulation_points.csv  
Персонажи, являющиеся точками сочленения сети.

kcore_values.csv  
Значения k-core для всех персонажей.

groups_all.csv  
Распределение персонажей по сообществам во всей сети.

groups_volume1.csv  
Распределение персонажей по сообществам в первом томе.

network_top25.png  
Визуализация сети из 25 наиболее центральных персонажей.

groups_volume1.png  
Визуализация групп персонажей в первом томе.

# Визуализация сети

В проекте строятся два основных графа.

Сеть топ-25 персонажей  
Отображает наиболее центральных персонажей произведения. Размер вершины соответствует взвешенной степени персонажа. Сообщества определяются алгоритмом Walktrap.

Группы персонажей в первом томе  
Отдельная сеть взаимодействий персонажей первого тома, позволяющая выявить основные сюжетные группы.

# Как запустить код

1. Установить необходимые библиотеки R:

install.packages("xml2")  
install.packages("tidyverse")  
install.packages("igraph")

2. Поместить в одну папку файлы:

War_and_Peace.xml  
war-and-peace.r  

3. Запустить скрипт в R:

source("war-and-peace.r")

После выполнения будут автоматически сгенерированы таблицы с результатами анализа и изображения сетевых графов.
